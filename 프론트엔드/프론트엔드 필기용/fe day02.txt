CSS

선택자 : 엘리먼트 선택자
엘리먼트의 이름으로 선택을 함
HTML 문서 내애서 스타일을 사용할때는 스타일 태그에 작성을 하면 됨


text-decoration: line-through;(취소선)
li 안에 넣으면 모든 li에 취소선이 그어짐

class 선택자
클래스를 지정해서 선택을함
점을 찍은 후 클래스 이름을 넣어준다.
클래스 이름은 아무렇게나 가능하다.(가눙하면 어떤 속성을 적용시켰는지 한눈에 알수있게 하는게 좋음)
한개의 엘리먼트에 여러개의 클래스를 추가 가능

id 선택자
중복이 되면 안된다.
같은 엘리먼트가 같은 id 공유 불가
하나의 엘리먼트가 여러개의 id 가질수 없음
#(해쉬)id 구조로 id를 만들 수 있다.

children 선택자
부모 엘리먼트의 자식을 선택함
부모를 선택하고 한칸 띄고 자식을 선택하면 된다.
ex) #list2 li {

}

pseudo 클래스
특별한 상태의 엘리먼트를 선택한다

pseudo 엘리먼트
가상의 엘리먼트를 만들고 선택한다

형제 선택자
앞에 선택한 엘리먼트 + 다음 엘리먼트를 선택
ex) 스타일을 #first +li로 했다면
first가 달린 엘리먼트 다음의 엘리먼트를 선택


CSS 단위
px = 픽셀
rem = 브라우저의 기본 글자크기 (사용 환경에 따라 다름) (root element)
em = 부모 엘리먼트 기준 글자크기 ()
percent = 100% = 부모 엘리먼트의 글자크기
vw = 100vw = view port의 넓이(width) (브라우저의 화면)
vh = 100vh = view port의 높이(height) (브라우저의 화면)

개발자 도구는 뷰포트에 포함되지 않음
브라우저의 화면만 포함
